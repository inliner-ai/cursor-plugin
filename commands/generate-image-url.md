# Generate Image URL

Use this when you want a fast, code-first result.

1. Ask for `project`, `description`, `width`, `height`, and optional `format`.
2. Call `generate_image_url`.
3. Return:
   - URL
   - HTML `<img>` snippet
   - Suggested alt text
